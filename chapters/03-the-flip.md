# ⚡ 零号翻转 | The Flip

> **适用场景：** GEX 穿越零点 | 市场状态切换 | 变盘信号

---

## 🎯 核心理念

想象一个**跷跷板**：

- 中间的支点就是 **GEX = 0** 的位置
- 一侧是正 Gamma（橡皮筋），另一侧是负 Gamma（冰面）
- 当跷跷板翻转时，游戏规则完全改变

```
        正 Gamma (震荡)
             ↑
    ─────────●─────────  ← GEX = 0 (翻转点)
             ↓
        负 Gamma (趋势)
```

**翻转的意义：** 市场从"均值回归"模式切换到"趋势跟踪"模式，或反过来。

---

## 📊 识别信号

### 从正转负 (🔵 → 🔴)

这通常是**危险信号**，预示着可能的大跌：

| 阶段 | 表现 |
|-----|------|
| 预警 | 价格接近 Put Wall，GEX 开始下降 |
| 触发 | 价格跌破 Put Wall，GEX 转负 |
| 确认 | 下跌加速，没有像样的反弹 |

```
价格走势：
     ────────────────
                     \
                      \  ← Put Wall 破位
                       \
                        ────────────
                               ↓
                        GEX 翻负，趋势开始
```

### 从负转正 (🔴 → 🔵)

这通常是**企稳信号**，预示着趋势可能结束：

| 阶段 | 表现 |
|-----|------|
| 预警 | 价格在某水平反复测试，GEX 开始上升 |
| 触发 | GEX 转正，价格不再创新低/新高 |
| 确认 | 价格开始区间震荡 |

## 🔍 GexBot 中如何观察翻转

### 关键监控指标

1. **Total GEX 数值**
   - 接近 0 → 进入翻转区域
   - 从正转负 / 从负转正 → 翻转确认

2. **Gamma Flip Level（翻转点位）**
   - GexBot 会显示预估的翻转价位
   - 价格穿越这个点位 = 可能触发翻转

3. **期权到期日分布**
   - 末日期权占比高时，翻转更容易发生
   - 周五下午翻转概率增加

### 翻转预警检查清单

```
□ Total GEX 接近 0？
□ 价格接近 Gamma Flip Level？
□ Put Wall 或 Call Wall 即将被突破？
□ 今天是周五（0DTE 到期日）？
□ 有重大事件（FOMC、CPI）？
```

如果勾选 3 个以上 → 高度警惕翻转！

## ⚔️ 战术执行

### 翻转前：准备阶段

**情况 A：你持有「橡皮筋」仓位，预感即将翻负**

1. 缩小仓位规模
2. 收紧止损
3. 不再逆势加仓

**情况 B：市场已在趋势中，预感即将翻正**

1. 开始逐步获利了结
2. 不再追涨杀跌
3. 准备切换到震荡策略

### 翻转时：关键操作

```
翻转确认检查清单：
□ GEX 已经穿越 0 点
□ 价格确认新状态（不是假突破）
□ 至少等待 15-30 分钟确认

确认后：
→ 平掉旧策略的仓位
→ 按新状态的策略入场
```

### 翻转后：新游戏开始

| 翻转方向 | 新策略 | 参考章节 |
|---------|--------|---------|
| 正 → 负 | 冰面竞速（顺势） | [🔴 第二章](./02-ice-surface.md) |
| 负 → 正 | 橡皮筋战术（逆势） | [🔵 第一章](./01-rubber-band.md) |

## ⚠️ 翻转陷阱

### 假翻转 (Fake Flip)

最危险的情况：GEX 短暂转负/转正，然后又翻回去。

```
真翻转：
GEX: +5B → +2B → 0 → -3B → -5B → 持续负值
                ↑
              翻转点

假翻转：
GEX: +5B → +1B → -0.5B → +2B → +4B
                    ↑
              假翻转！价格又被吸回
```

**如何避免假翻转陷阱？**

1. **时间确认**：等待至少 30 分钟
2. **价格确认**：看价格是否真的脱离了原来的区间
3. **幅度确认**：GEX 是否明显进入负/正区域

### 尾盘翻转

收盘前 1 小时的翻转要格外小心：

- ✅ 可能是真翻转（为第二天定调）
- ❌ 可能是收盘前的技术性波动

**建议**：尾盘翻转不急于入场，等第二天开盘确认。
