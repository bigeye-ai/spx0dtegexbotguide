# 🔵 橡皮筋战术 | Rubber Band Strategy

> **适用场景：** 正 Gamma 环境 (GEX > 0) | 粘性市场 | 震荡行情

---

## 🎯 核心理念

想象一根**橡皮筋**绑在一个固定点上：

- 你把它拉开，它会**弹回来**
- 拉得越远，弹回的力量越大
- 它不会轻易脱离那个固定点

在正 Gamma 环境下，SPX 价格就像这根橡皮筋：

```
价格上涨 → 做市商卖出 → 压制上涨
价格下跌 → 做市商买入 → 支撑下跌
        ↓
    价格被"吸回"中心
```

---

## 📊 识别信号

### GexBot 数据特征

| 指标 | 正 Gamma 信号 |
|-----|--------------|
| Total GEX | **> 0**（正值） |
| GEX 强度 | 数值越大，"橡皮筋"越紧 |
| 关键行权价 | 大量 Call OI 集中的位置 = 阻力 |
|            | 大量 Put OI 集中的位置 = 支撑 |

### 市场表现

- ✅ 价格在一个区间内反复震荡
- ✅ 突破尝试失败，快速回归
- ✅ 日内波动率下降
- ✅ VIX 相对平稳

---

## ⚔️ 战术执行

### 基本策略：逢高卖、逢低买

```
          阻力位 (Call Wall)
              ↓
     ────────●────────  ← 在这里做空
            /  \
           /    \
          /      \
     ────●────────●────  ← 在这里做多
              ↑
          支撑位 (Put Wall)
```

### 具体步骤

1. **确认环境**
   - 打开 GexBot，确认 Total GEX > 0
   - 识别主要的 Call Wall 和 Put Wall

2. **等待价格接近边界**
   - 价格接近 Call Wall → 准备做空
   - 价格接近 Put Wall → 准备做多

3. **入场**
   - 等待反转 K 线确认（如：锤子线、吞没形态）
   - 不要在边界"接飞刀"，要等确认

4. **止损与止盈**
   - 止损：放在边界外侧 5-10 点
   - 止盈：对面边界的 50%-70%

---

## ⚠️ 风险提示

### 这个策略会失效的情况

1. **GEX 翻负**：橡皮筋断了，变成冰面竞速
2. **重大新闻**：FOMC、CPI 等事件可能打破平衡
3. **临近收盘**：Gamma 效应减弱

### 常见错误

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| 在边界追涨杀跌 | 等反转确认再入场 |
| 忽视 GEX 变化 | 实时监控，随时调整 |
| 止损设太近 | 给价格一些呼吸空间 |
