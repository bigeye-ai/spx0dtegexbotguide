# 🔴 冰面竞速 | Ice Surface Strategy

> **适用场景：** 负 Gamma 环境 (GEX < 0) | 滑性市场 | 趋势行情

---

## 🎯 核心理念

想象你站在一块**光滑的冰面**上：

- 轻轻一推，你就会**滑很远**
- 没有摩擦力阻止你
- 一旦开始滑动，很难停下来

在负 Gamma 环境下，SPX 价格就像冰面上的冰球：

```
价格上涨 → 做市商被迫买入对冲 → 加速上涨
价格下跌 → 做市商被迫卖出对冲 → 加速下跌
        ↓
    趋势自我强化
```

---

## 📊 识别信号

### GexBot 数据特征

| 指标 | 负 Gamma 信号 |
|-----|--------------|
| Total GEX | **< 0**（负值） |
| GEX 强度 | 负值越大，"冰面"越滑 |
| Put Wall 破位 | 下跌趋势加速的催化剂 |
| Call Wall 破位 | 上涨趋势加速的催化剂 |

### 市场表现

- ✅ 价格单边运行，回调浅
- ✅ 突破后快速远离突破点
- ✅ 日内波动率上升
- ✅ VIX 飙升（尤其下跌时）

---

## ⚔️ 战术执行

### 基本策略：顺势而为，不要抄底逃顶

```
    趋势方向
        ↓
    ────────────────→
         /
        /   ← 回调买入点
       /
    ──/─────────────→
           
    ❌ 不要在这里反向操作
```

### 具体步骤

1. **确认环境**
   - 打开 GexBot，确认 Total GEX < 0
   - 判断主要趋势方向

2. **等待回调入场**
   - 趋势向上 → 等小回调后做多
   - 趋势向下 → 等小反弹后做空

3. **入场时机**
   - 回调到短期均线（如 9EMA、21EMA）
   - 出现趋势延续 K 线形态

4. **止损与止盈**
   - 止损：前一个摆动低点/高点
   - 止盈：不设固定目标，用移动止损跟踪

### 移动止损技巧

```
入场价：5800
趋势：向上

波动过程：
5800 → 5820 → 5810 → 5840 → 5830 → 5860
                          ↑
                    移动止损到 5810
                    
5860 → 5880 → 5870 → 5900
                ↑
          移动止损到 5850
```

---

## ⚠️ 风险提示

### 这个策略的核心风险

1. **趋势反转**：负 Gamma 不是永恒的
2. **假突破**：需要配合成交量确认
3. **过度杠杆**：趋势明确≠可以梭哈

### 负 Gamma 的陷阱

| 陷阱 | 表现 | 应对 |
|-----|------|------|
| V 型反转 | 单边暴跌后急速反弹 | 等 GEX 变正再反手 |
| 尾盘波动 | 收盘前 30 分钟剧烈波动 | 提前减仓或平仓 |
| 缺口风险 | 隔夜大幅高开/低开 | 不持仓过夜 |

## 🔄 与橡皮筋战术的切换

```
橡皮筋 (正Gamma)              冰面竞速 (负Gamma)
     ↑                              ↑
     |          零号翻转            |
     └──────────  ⚡  ──────────────┘
              GEX = 0
```

**关键转换信号：**
- 正 → 负：Put Wall 被跌破 + GEX 转负
- 负 → 正：价格稳定在某个水平 + GEX 转正

详见：[⚡ 零号翻转](./03-the-flip.md)
