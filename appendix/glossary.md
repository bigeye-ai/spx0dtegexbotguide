## 📊 期权基础术语

| 术语 | 英文 | 定义 |
|-----|------|------|
| **期权** | Option | 在特定时间以特定价格买卖标的资产的权利 |
| **看涨期权** | Call | 买入标的的权利 |
| **看跌期权** | Put | 卖出标的的权利 |
| **行权价** | Strike Price | 期权约定的交易价格 |
| **到期日** | Expiration | 期权失效的日期 |
| **权利金** | Premium | 购买期权支付的价格 |
| **持仓量** | Open Interest (OI) | 未平仓的期权合约数量 |

---

## 🔢 希腊字母 (Greeks)

| 术语 | 符号 | 定义 | 简单理解 |
|-----|------|------|---------|
| **Delta** | Δ | 标的价格变动对期权价格的影响 | 期权"跟涨跟跌"的程度 |
| **Gamma** | Γ | 标的价格变动对 Delta 的影响 | Delta 的"加速度" |
| **Theta** | Θ | 时间流逝对期权价格的影响 | 时间杀手，每天亏多少 |
| **Vega** | ν | 波动率变动对期权价格的影响 | 波动率敏感度 |

---

## 📈 GEX 相关术语

| 术语 | 定义 |
|-----|------|
| **GEX** | Gamma Exposure，做市商因对冲期权产生的买卖压力 |
| **Total GEX** | 所有期权的 GEX 之和，正值=震荡，负值=趋势 |
| **正 Gamma** | GEX > 0，做市商做多 Gamma，市场倾向震荡 |
| **负 Gamma** | GEX < 0，做市商做空 Gamma，市场倾向趋势 |
| **Call Wall** | 最大 Call OI 集中的行权价，潜在阻力 |
| **Put Wall** | 最大 Put OI 集中的行权价，潜在支撑 |
| **Gamma Flip** | GEX 从正转负或从负转正的价格水平 |
| **HVL** | High Volume Level，高成交量价位 |

## 🕐 时间相关术语

| 术语 | 英文 | 定义 |
|-----|------|------|
| **0DTE** | Zero Days to Expiration | 当天到期的期权 |
| **末日期权** | - | 0DTE 的俗称 |
| **OPEX** | Options Expiration | 期权到期日，通常指每月第三个周五 |
| **日内交易** | Day Trading / Intraday | 当天开仓当天平仓的交易方式 |

## 💼 交易术语

| 术语 | 英文 | 定义 |
|-----|------|------|
| **做多** | Long / Buy | 买入，预期上涨 |
| **做空** | Short / Sell | 卖出，预期下跌 |
| **止损** | Stop Loss | 预设的亏损退出点 |
| **止盈** | Take Profit | 预设的盈利退出点 |
| **对冲** | Hedge | 用相反头寸降低风险 |
| **做市商** | Market Maker | 提供流动性的机构交易者 |

## 📉 市场状态术语

| 术语 | 定义 |
|-----|------|
| **震荡** | 价格在区间内反复波动，无明确趋势 |
| **趋势** | 价格朝一个方向持续运动 |
| **突破** | 价格越过关键支撑/阻力位 |
| **假突破** | 突破后快速回落，未能延续 |
| **回调** | 趋势中的反向修正 |
| **支撑** | 阻止价格下跌的价位 |
| **阻力** | 阻止价格上涨的价位 |
